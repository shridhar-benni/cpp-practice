## 🚀 Getting Started
### Prerequisites
- Install a C++ compiler (e.g., GCC, Clang, or MSVC)
- (Optional) Install an IDE like VS Code, CLion, or Code::Blocks

### Compilation and Execution
To compile and run a C++ program:
```sh
 g++ *.cpp -o output
 ./output
```
